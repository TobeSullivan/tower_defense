# Steam fests — Wend marketing calendar

Captured 2026-06-11. Concrete dated reference so we don't re-research. Pairs with `gtm.md`

## The split that governs everything

- **Next Fest = the pre-launch discovery event.** Needs a public Coming Soon page + a publicly
  playable demo live by fest start; game must release *after* the fest. **You get exactly ONE
  Next Fest, ever** — choose deliberately.
- **Themed genre fests = sale events.** Steam *auto-invites* games whose primary store tags
  match the theme; you opt in + set a discount. Built around discounting a **released** product.
  So the genre-perfect ones (Tower Defense, PvP) are **launch-or-later** levers, not pre-launch.
- **Seasonal sales** (Spring/Summer/Autumn/Winter) are open to all but only matter once you
  have a priced, released game; front-page slots skew to proven top-sellers.
